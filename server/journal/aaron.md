# Aaron

## Overview

Aaron is a student in Bournemouth who is beginning Sixth Form, studying Maths, Physics and Cyber Security. He enjoys understanding systems deeply rather than just using them, and prefers to learn by building real projects.

## Primary Project

Aaron is the creator of Swyft, a lightweight, local-first desktop environment designed to make older hardware more useful. Swyft is built around openness, simplicity and user choice rather than lock-in or aggressive AI integration.

Patrick is intended to become Swyft's optional AI assistant.

## Core Principles

- AI must always be optional.
- Local-first wherever practical.
- Open standards over vendor lock-in.
- Build tools that remove friction rather than demand attention.
- Users should always remain in control.

## Development Style

Aaron prefers understanding why something works before implementing it.

He values clean architecture and modular design over quick hacks.

He likes iterative development where each feature is fully understood before moving on.

## Current Goals

- Build Patrick Core.
- Integrate Patrick with Obsidian.
- Integrate Patrick with VS Code.
- Eventually integrate Patrick directly into Swyft Desktop.

## Obsidian

Obsidian acts as Patrick's long-term knowledge base.

Project documentation, research, school notes and design decisions are stored there.

Patrick should search Obsidian before relying solely on model knowledge whenever possible.

## Interaction Style

Patrick should behave like a long-term collaborator.

He should challenge assumptions, explain reasoning, remember ongoing projects and help improve ideas rather than simply answer questions.

Humour is welcome, but accuracy comes first.
